### 이거 어때?의 차별점

- 사용자의 소비 패턴은 다양한 요인에 영향을 받지만 기존의 서비스들에서는 오로지 소비 패턴만을 분석하여 그에 맞는 카드와 카드 혜택을 제공하거나 사용자가 직접 원하는 카드 혜택을 찾아야 했으므로 다양하고 자동화된 추천을 제공하지 못했습니다. 이거 어때?는 이와 다르게 사용자의 복합적 요인을 AI를 통해 분석하여 다양하고 자동화된 카드 혜택 및 혜택을 누릴 수 있는 가맹점 추천 서비스를 제공한다는 차별점을 가지고 있습니다. 

<br>

## ⚙️ Use Case
![예시-케이스](https://github.com/vuswltmd/ClimateShoppingTrends/assets/101254480/1a3323ed-57ee-45dd-902f-9722eca54baa)

>1. http://127.0.0.1:5000에 접속합니다
>2. 본인의 정보를 입력합니다. 선호도, 분위기 선호도, 자차 유무는 음식관련 추천에 적용되는 항목입니다.
>3. AI 모델이 사용자 입력 정보를 분석하여 관련 업종을 3가지 추천해줍니다.
>4. 1순위 ~ 3순위에 해당하는 업종에 대해서 자세히 보기를 클릭하면 해당 업종과 관련된 장소 및 카드혜택을 제공합니다.
